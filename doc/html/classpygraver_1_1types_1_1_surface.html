std::vector< std::shared_ptr< Path > > pygraver::types::Surface::correct_height ( const std::vector< std::shared_ptr< Path > > &  paths,
const double  clearance,
const double  safe_height,
const bool  outside = true,
const bool  fix_contours = false 
) const

Correct height of given paths.

I wrote this function with fabrication in mind. For a number of operations (milling, engraving, turning, ...), we need enough clearance to pass the tool; besides, it needs to be made to follow the surface. This function adapts the height of the path points for this purpose, based on whether they are inside or outside the surface.

Parameters
pathsa collection of paths to correct.
clearanceclearance near a contour.
safe_heightheight in exclusion area (outside or inside).
outsideif true, exclusion area is outside surface; if false, exclusion area is inside.
fix_contoursif true, adds supplementary points on contours to increase accuracy.
Returns
a collection of corrected paths.
